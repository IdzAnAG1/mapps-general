# Auth Service

Отвечает за регистрацию пользователей и выдачу JWT токенов.

## Что делает

- Регистрация пользователя (email + password + username)
- Авторизация — проверка пароля, выдача JWT
- Хранение пользователей в PostgreSQL

## Технологии

- Go + Kratos
- PostgreSQL
- JWT (HS256)
- Goose — миграции БД

## gRPC методы

- `Register` — создать пользователя
- `Login` — войти, получить токен
